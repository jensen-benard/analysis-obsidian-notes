We define set $A$ to be any unordered collection of objects.

If $x$ is an object, we say that $x$ is an element of $A$ or $x \in A$ if $x$ lies in the collection; otherwise we say $x \notin A$.

For example,
$$3 \in \{1, 2, 3, 4, 5\}$$
but
$$ 7 \notin \{1, 2, 3, 4, 5\}$$
